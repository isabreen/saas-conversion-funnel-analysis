# saas-conversion-funnel-analysis
Data-driven analysis of a B2B SaaS trial-to-paid funnel to identify user drop-off points and growth opportunities.

The Goal: Analyze the journey from Website Visit to Subscription.

The Finding: A 60% drop-off at the "Onboarding" stage is the primary growth blocker.

The Impact: Reducing this drop-off by 10% could lead to a significant revenue lift.
